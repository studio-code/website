What you would expect

Deep integration

StudIO has great integration with services you already know and use: GitHub, Gists, Gitlab, ... That mean you won't change your habits by switching.

Native

StudIO is native but hackable: almost everything in the app is written in pure Swift. But our editor is compatible with most CodeMirror plugins (see docs for integration).

Easy to use

All you need to get started is to download the app. Everything else is intuitive and simple. StudIO aims to reproduce the desktop experience on portable devices.

Upgradable

StudIO receives constant updates with incredible new features. The future has never been as bright.

And much more

Plugins

StudIO is built on top of a modified CodeMirror that supports community created plugins to improve your and everyone else's experience.

No compromises

StudIO is compatible with the new Files app, which means you can directly import your code into other apps such as iSH for using a full Linux powered terminal on your device.

Fun to use

When using an iPad (or other devices), creating something should be fun. StudIO aims above all to let developers and enthusiasts benefit from the most enjoyable experience.

Cheap

While other apps like Textastic or Buffer editor costs around 10$, StudIO only costs 4.99$. Price shouldn't be a feature, it should be a strength.
